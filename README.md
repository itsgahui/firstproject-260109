## English

This is my first fully completed responsive landing page.  
It was built using only HTML and CSS, with a focus on layout structure, responsive design, and accessibility.

### Tech Stack
- HTML5
- CSS3 (Flexbox)
- Media Queries

### Features
- Responsive layout for Desktop and Mobile
- Layout design based on Flexbox
- Hover effects
- Use of ARIA attributes with accessibility in mind

### Responsive Design
- Desktop: Horizontal (side-by-side) layout
- Mobile: Vertical (stacked) layout

### What I Learned
- How to design layouts using Flexbox
- Responsive text sizing using vw units
- Handling mobile layouts with media queries
- The relationship between UI details and user experience

### Acknowledgments
This project is based on a Frontend Mentor challenge.


## Français

Il s’agit de ma première page d’accueil responsive entièrement finalisée.  
Elle a été réalisée uniquement avec HTML et CSS, en mettant l’accent sur la structure du layout, le design responsive et l’accessibilité.

### Technologies utilisées
- HTML5
- CSS3 (Flexbox)
- Media Queries

### Fonctionnalités
- Mise en page responsive pour ordinateur et mobile
- Conception du layout basée sur Flexbox
- Effets hover
- Utilisation des attributs aria en tenant compte de l’accessibilité

### Design Responsive
- Ordinateur : mise en page horizontale (côte à côte)
- Mobile : mise en page verticale (empilée)

### Ce que j’ai appris
- Concevoir des mises en page avec Flexbox
- Gérer la taille du texte de manière responsive avec les unités vw
- Adapter les interfaces mobiles grâce aux media queries
- L’impact des détails de l’interface sur l’expérience utilisateur
